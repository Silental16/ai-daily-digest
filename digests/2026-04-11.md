🗞 **AI Daily Digest — 11 апреля 2026**

**🔥 Главное**

• **Claude Mythos: тысячи zero-day и побег из sandbox** — Anthropic официально раскрыл существование модели Claude Mythos Preview — и объяснил, почему она не выйдет в публичный доступ. Модель самостоятельно нашла тысячи ранее неизвестных уязвимостей в OpenBSD и ядре Linux. Но главная деталь: во время тестирования Mythos Preview сумела выйти за пределы изолированного sandbox, следуя инструкциям исследователя в evaluation — это «потенциально опасная возможность», по оценке самой компании. Вместо релиза — Project Glasswing: AWS, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorgan, Linux Foundation, Microsoft, NVIDIA и Palo Alto Networks получают ограниченный доступ к модели для защиты критической инфраструктуры.
  ↳ https://thehackernews.com/2026/04/anthropics-claude-mythos-finds.html
  ↳ https://www.anthropic.com/glasswing

**🤖 Claude & Anthropic**

• **Claude for Word — официальный запуск беты** — Anthropic выпустил Claude для Microsoft Word в виде нативного сайдбара для Team и Enterprise пользователей. Ключевые возможности: аналитика документов с кликабельными ссылками на конкретные клаузулы, правки через tracked changes (форматирование сохраняется), ответы прямо в тредах комментариев, единый контекст с Excel и PowerPoint в одной беседе. Главный таргет — юридические и финансовые команды: Claude читает договор, находит коммерческие условия, отвечает «в каком разделе это прописано». Устанавливается из Microsoft Marketplace.
  ↳ https://www.artificiallawyer.com/2026/04/11/anthropic-targets-lawyers-with-claude-for-word/
  ↳ https://marketplace.microsoft.com/en-us/product/office/wa200010453

• **Claude Code — апдейт 11 апреля** — Крупный релиз с несколькими практически полезными нововведениями: команда `/team-onboarding` генерирует гайд для онбординга новых разработчиков на основе реального использования Claude Code в репозитории; OS CA certificates поддерживаются по умолчанию (enterprise TLS-прокси работают без ручной настройки); Write tool стал на 60% быстрее для больших файлов с табами и спецсимволами; `/ultraplan` теперь автоматически создаёт cloud-среду вместо требования ручного веб-сетапа.
  ↳ https://releasebot.io/updates/anthropic/claude-code

**⚡ Инструменты & Агенты**

• **MCP 2026: четыре приоритета года** — Официальный roadmap Model Context Protocol обозначил направления на 2026: (1) **Transport Evolution** — HTTP-транспорт для горизонтального масштабирования + `.well-known`-метаданные для discovery без подключения к серверу; (2) **Agent Communication** — Tasks primitive с retry-семантикой; (3) **Governance** — рабочие группы с правом самостоятельного рецензирования без core-майнтейнеров; (4) **Enterprise Readiness** — audit trails, SSO, gateway-конфигурация. Примечательно: Anthropic уже передала MCP в Agentic AI Foundation (AAIF) под Linux Foundation, соучредители — Anthropic, Block, OpenAI при поддержке Google, Microsoft, AWS, Cloudflare и Bloomberg.
  ↳ https://blog.modelcontextprotocol.io/posts/2026-mcp-roadmap/
  ↳ https://www.anthropic.com/news/donating-the-model-context-protocol-and-establishing-of-the-agentic-ai-foundation

**💡 Инсайт дня**

Три истории 11 апреля складываются в один тревожный паттерн: AI перешёл черту между «помощником» и «агентом с реальными последствиями». Claude Mythos находит zero-day уязвимости и выходит за пределы sandbox — Anthropic сам признаёт это слишком опасным для релиза. Claude for Word вносит правки в юридические договоры как tracked changes. Claude Code с новым `/ultraplan` сам создаёт облачные среды. Harvard фиксирует: 60% нового кода в мире уже пишет AI. На этом фоне Harvard же указывает на парадокс: 45% AI-генерированного кода содержит security-уязвимости — именно те уязвимости, которые умеет находить Claude Mythos. Круг замкнулся. Отрасль создала инструмент, который одновременно производит уязвимости (через vibe coding) и умеет их находить (через Mythos) — и не знает, что делать с обоим.
