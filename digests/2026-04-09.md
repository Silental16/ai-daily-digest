🗞 **AI Daily Digest — 9 апреля 2026**

**🔥 Главное**

• **Anthropic обошёл OpenAI по выручке — $30B ARR** — Anthropic впервые вышел вперёд: $30 млрд годового run-rate против $24–25 млрд у OpenAI. Рост с $9 млрд произошёл всего за 4 месяца, с $1 млрд — за 15. 80% выручки от enterprise-клиентов, 1000+ компаний тратят >$1 млн/год (удвоились за <2 месяца). Потенциальное IPO в октябре 2026. При этом Anthropic тратит в 4 раза меньше на обучение моделей, чем OpenAI.
  ↳ https://www.the-ai-corner.com/p/anthropic-30b-arr-passed-openai-revenue-2026

• **Claude Mythos: слишком мощный для публикации** — Anthropic объявил Project Glasswing и раскрыл возможности своей нераскрытой модели Claude Mythos. В тестах Mythos нашёл тысячи уязвимостей высокой степени серьёзности во всех крупных ОС и браузерах, включая 27-летнюю дыру в OpenBSD и несколько уязвимостей в ядре Linux, дающих полный контроль над машиной. Anthropic не публикует модель — работает с Apple и другими компаниями как defensive cybersecurity tool.
  ↳ https://globalnews.ca/news/11769446/anthropic-ai-model-too-powerful/

• **Meta Muse Spark — нативно мультимодальная reasoning-модель** — Meta Superintelligence Labs (под руководством Alexandr Wang) выпустила Muse Spark: поддержка tool-use, visual chain-of-thought, multi-agent orchestration. Разворачивается в WhatsApp, Instagram, Facebook, Messenger и AI-очках Meta. Доступ через API для select-партнёров.
  ↳ https://renovateqr.com/blog/ai-models-april-2026

  ↳ https://letsdatascience.com/news/anthropic-launches-managed-claude-agents-for-enterprises-55705fb3

  ↳ https://news.bitcoin.com/federal-judges-deny-anthropic-relief-in-claude-military-ai-ban-set-may-oral-arguments/

• **Claude Code — мажорный апдейт** — Focus view (Ctrl+O): минималистичный режим отображения с одной строкой summary инструмента и diff-статистикой. Скорость вычисления diff для Write tool выросла на 60%. Новый refreshInterval для status line, индикатор "● N running" агентов в /agents, улучшена стабильность MCP и sandbox в Linux.

**💡 Инсайт дня**

Апрель 2026 — это неделя, когда стало очевидно: Anthropic больше не underdog. Компания одновременно обошла OpenAI по выручке, запустила enterprise-инфраструктуру для агентов (Managed Agents), подтвердила лидерство Claude Code как #1 coding-инструмента (46% у разработчиков против 19% у Cursor), вывела на рынок так называемый Project Glasswing — и при этом судится с Пентагоном за право работать с правительством США. Модель Claude Mythos, настолько мощная, что её не рискуют публиковать, — это одновременно PR-ход и реальная техническая заявка на лидерство в безопасности. Anthropic превратился из "более безопасной альтернативы OpenAI" в самостоятельную enterprise AI-платформу с собственной орбитой. А вся индустрия тем временем тихо переходит от "я промпчу и надеюсь" к реальной agentic engineering — с агентами, которые платят картой, пишут код командами из 5 штук и находят zero-days в Linux.
