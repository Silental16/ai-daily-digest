**💡 Инсайт дня**

История с Claude Code и Pro-планом — не просто pricing-фиаско, а зеркало в которое смотрит вся индустрия. Anthropic задеплоил лучший coding-агент, включил его в $20/мес и обнаружил, что usage взрывной настолько, что инфраструктура трещит. Теперь компания пытается монетизировать это без отчуждения developer-комьюнити, которое сделало Claude Code вирусным. Параллельно — двойные compute-сделки: AWS ($100 млрд, 5 ГВт) + Google/Broadcom ($3,5 ГВт) суммарно обязывают Anthropic к $130+ млрд трат. Такой capex требует соответствующей ARR-машины. MCP-уязвимость добавляет давление: чем глубже Claude Code и экосистема MCP встраиваются в production, тем выше ставки от security-инцидентов. Ближайшие 90 дней покажут, пойдёт ли Anthropic по пути поднятия цен снизу (Pro → Max) или создаст отдельный developer-tier. Обратный отсчёт запущен.
