🗞 **AI Daily Digest — 2 мая 2026**

**🔥 Главное**

• **Claude Security — публичная бета для всех Enterprise-клиентов** — Anthropic открыл Claude Security: end-to-end сканер уязвимостей кода на базе Opus 4.7, работающий командой параллельных агентов. Система изучает data flows по всей кодовой базе, находит уязвимости — в том числе те, которые традиционные SAST/DAST-сканеры пропускали годами — и генерирует патчи для ревью. Никакого кастомного tooling: подключаешь репозиторий через claude.com. С февральского preview сотни организаций уже нашли реальные дыры в production-коде. Партнёрства с CrowdStrike, Microsoft Security, Palo Alto Networks, SentinelOne, TrendMicro и Wiz.
  ↳ https://siliconangle.com/2026/04/30/anthropic-announces-claude-security-public-beta-find-fix-software-vulnerabilities/
  ↳ https://thenewstack.io/anthropics-claude-security-beta/

**🤖 Claude & Anthropic**

• **Hackathon «Built with Opus 4.6»: результаты** — Anthropic опубликовал победителей первого официального Claude Code hackathon. Из 13 000+ заявок отобраны 500 участников — одна неделя и API-кредиты. Топовые работы: Jon собрал Elisa (образовательное приложение для детей) — 76 коммитов, 39 000+ строк кода, 1 500 тестов за 30 часов, полный продукт за 6 дней; нетехнический project manager — приложение управления стрессом за 6 недель без опыта разработки; Михал Недошытко — PostVisit, healthcare-инструмент интеграции медкарт, собранный за поездку из Брюсселя в Сан-Франциско.
  ↳ https://claude.com/blog/meet-the-winners-of-our-built-with-opus-4-6-claude-code-hackathon

**⚡ Инструменты & Агенты**

• **MCP: обновления по всему стеку** — 2 мая обновились ключевые компоненты экосистемы Model Context Protocol: официальный Ruby SDK, MCP Inspector (visual testing tool для отладки серверов) и community registry. Ruby-поддержка закрывает одну из последних брешей для enterprise Rails-разработчиков. Экосистема MCP насчитывает уже тысячи официальных и community-серверов.
  ↳ https://github.com/modelcontextprotocol

• **Claude Code v2.1.126 — ключевые изменения** — Релиз от 1 мая продолжает набирать обсуждение: `/model` picker теперь подтягивает список моделей с gateway через `ANTHROPIC_BASE_URL`; новая `claude project purge [path]` полностью сбрасывает состояние проекта (транскрипты, tasks, file history, config) — поддерживает `--dry-run`; security-фикс: `allowManagedDomainsOnly` больше не игнорировался молча; OAuth-авторизация через paste-код в терминале (критично для WSL2, SSH и контейнеров).
  ↳ https://code.claude.com/docs/en/changelog

**🌐 Индустрия**

• **GPT-5.5 дотянулся до Mythos в кибербезопасности** — Публично доступный GPT-5.5 показал результаты на уровне закрытого Anthropic Mythos Preview в задачах offensive cybersecurity: оба решили 32-шаговую симуляцию атаки на корпоративную сеть — первые LLM, справившиеся с этим тестом. Это прямо бьёт по позиционированию Mythos как «ответственно ограниченного» инструмента для пентестеров: если те же возможности доступны через публичное API конкурента, вопрос контроля выходит за рамки одной компании.
  ↳ https://blog.mean.ceo/ai-news-may-2026/

• **Agentic Engineering: реакция индустрии на тезис Карпати** — Выступление Карпати на AI Ascent (1 мая) вызвало волну разборов: Analytics Drift, The New Stack, DEV Community публикуют анализ «Software 3.0». Ключевые тезисы: декабрь 2025 — точка инверсии, когда Карпати перешёл с 80% самостоятельного кода на 80% делегирования агентам; «vibe coding raises the floor, agentic engineering preserves the ceiling» — разница между демократизацией доступа и сохранением качества.
  ↳ https://analyticsdrift.com/andrej-karpathy-agentic-engineering-software-3/
  ↳ https://thenewstack.io/vibe-coding-is-passe/

**💡 Инсайт дня**

Claude Security и результаты hackathon — два сигнала об одном сдвиге. AI перестаёт быть инструментом только для написания кода и становится инструментом для контроля его качества. Нетехнический PM, собравший приложение за 6 недель, и Jon с 39K строк за 30 часов — это разные точки входа в одну парадигму: человек ставит задачу, AI выполняет работу. Когда тот же AI начинает находить дыры в собственном же коде — круг замыкается. А появление GPT-5.5 на уровне Mythos в offensive security говорит о том, что frontier-возможности теперь коммодитизируются быстрее, чем компании успевают выстраивать вокруг них конкурентные рвы.
