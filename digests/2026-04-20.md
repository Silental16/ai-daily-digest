• **ChatGPT и Codex упали по всему миру** — около 10:05 утра по ET Downdetector зафиксировал волну жалоб, пик — более 13 000 репортов. Пользователи получали белые экраны, вечные спиннеры, ошибки входа. Под удар попали старые переписки (63% жалоб), аватентификация (27%), а также Codex, Voice Mode и генерация изображений. OpenAI поначалу признал лишь «деградацию производительности», затем апгрейдировал статус до «частичного сбоя» и задеплоил фикс — к 12:00 ET всё восстановилось. Суммарный даунтайм — около 90 минут.

• **Критическая уязвимость в сердце MCP** — OX Security опубликовала разбор архитектурного дефекта во всех официальных MCP SDK (Python, TypeScript, Java, Rust): STDIO-интерфейс запускает локальный серверный процесс командой, которая выполняется вне зависимости от того, стартовал ли сам сервер. Передай вредоносную команду — получи ошибку, но команда уже выполнилась. Под угрозой — более 150 млн загрузок, 200 000+ уязвимых инстансов, 7 000+ публичных серверов. Под удар попадают Cursor, VS Code, Windsurf, Claude Code и Gemini-CLI. Исследователи успешно выполнили RCE на шести live-production платформах, включая LiteLLM, LangChain и IBM LangFlow. Anthropic назвал это «expected behavior».
  ↳ https://thehackernews.com/2026/04/anthropic-mcp-design-vulnerability.html
  ↳ https://www.securityweek.com/by-design-flaw-in-mcp-could-enable-widespread-ai-supply-chain-attacks/

**🤖 Claude & Anthropic**

• **Claude Code v2.1.116 вышел 20 апреля** — самый плотный релиз за последние недели. Главное: `/resume` на больших сессиях стал до 67% быстрее, spinner при долгом думании показывает прогресс («still thinking» → «almost done thinking»), `/config` теперь ищет по значениям (ищешь «vim» — находит режим редактора), `/doctor` можно открыть прямо во время ответа модели. Из security-фиксов: закрыт обход sandbox auto-allow для опасных путей (`rm`/`rmdir` на `/`, `$HOME` и системных директориях). Также починили рендеринг шрифтов Devanagari и Indic-скриптов.
  ↳ https://github.com/anthropics/claude-code/releases

• **Claude Opus 4.7 — теперь в Amazon Bedrock** — модель доступна с 1M-токенным контекстом, скором 64.3% на SWE-bench Pro и 87.6% на SWE-bench Verified. AWS выпустил специальный Weekly Roundup, посвящённый этому релизу и одновременному GA AWS Interconnect.

**💡 Инсайт дня**

MCP за полтора года прошёл путь от Anthropic-эксперимента до де-факто инфраструктурного слоя AI-агентов — Claude Desktop и Cursor добавили полную поддержку MCP v2.1. И именно в этот момент уязвимость, зашитая в архитектуру протокола с самого начала, превращается из академической в критическую: 150 миллионов загрузок, 200 тысяч уязвимых инстансов. Ответ Anthropic «это expected behavior» переносит бремя security-санации на десятки тысяч downstream-разработчиков, многие из которых не являются security-инженерами. История MCP — точная копия истории любого стремительно ставшего стандартом протокола: скорость принятия обгоняет зрелость безопасности. Смотрите на это как на ранний Log4Shell для AI-эпохи.
