🗞 **AI Daily Digest — 25 апреля 2026**

**🔥 Главное**

• **Bloomberg написал про «самый вирусный график в AI» — и это не бенчмарк** — 25 апреля Bloomberg опубликовал разбор феномена OpenClaw: локальный личный AI-ассистент за несколько недель вырос с 9 000 до 210 000+ GitHub-звёзд — быстрее, чем любой другой open-source проект в истории платформы. OpenClaw работает полностью локально, соединяет LLM-модели с 50+ интеграциями (Telegram, WhatsApp, Slack, Discord, Signal) и не отправляет данные в облако. Статья Bloomberg ставит вопрос прямо: что говорит нам этот рост о настроениях пользователей, уставших от подписок и утечек данных?
  ↳ https://www.bloomberg.com/news/articles/2026-04-25/understanding-the-most-viral-chart-in-artificial-intelligence
  ↳ https://www.kdnuggets.com/openclaw-explained-the-free-ai-agent-tool-going-viral-already-in-2026

• **GPT-5.5 вышел для платных пользователей — и у него есть prompting guide** — OpenAI завершил rollout GPT-5.5 для подписчиков ChatGPT и Codex. Модель умеет самостоятельно планировать шаги, подбирать инструменты и проверять собственные ошибки до выдачи ответа. 25 апреля Simon Willison опубликовал разбор официального prompting guide: ключевые советы — меньше few-shot примеров (модель хорошо обобщает), явно указывать нежелательный формат, доверять автономии модели в выборе tools.
  ↳ https://simonwillison.net/2026/Apr/25/gpt-5-5-prompting-guide/

**🤖 Claude & Anthropic**

• **Rate Limits API вышел 25 апреля** — Anthropic выпустил программный API для запроса rate-лимитов: администраторы организаций теперь могут получать данные о лимитах своего workspace напрямую через API, без ручной проверки в консоли. Небольшое, но важное улучшение для команд, строящих мониторинг и автоматизацию вокруг Claude.
  ↳ https://releasebot.io/updates/anthropic

• **Claude Code 2.1.120 сломал --resume и --continue — и был откатан автоматически** — пользователи Claude Code обнаружили crash при попытке продолжить предыдущую сессию через флаги `--resume` или `--continue`. Команда откатила проблемный релиз до 2.1.119 через auto-update механизм. Инцидент показывает, что rate восстановления у Anthropic ускорился — в апреле баги фиксируются в часах, не в днях.
  ↳ https://claudefa.st/blog/guide/changelog
  ↳ https://releasebot.io/updates/anthropic/claude-code

• **Anthropic тестирует удаление Claude Code из Pro** — ограниченный эксперимент (~2% новых prosumer-подписчиков): Claude Code становится недоступен на $20/мес тарифе, стартует с Max 5x ($100/мес). Существующие подписчики Pro и Max не затронуты. Anthropic называет это «small test» — но раздражение в комьюнити высокое.
  ↳ https://pasqualepillitteri.it/en/news/1211/claude-code-removed-pro-plan-anthropic-april-2026
  ↳ https://www.theregister.com/2026/04/22/anthropic_removes_claude_code_pro/

**⚡ Инструменты & Агенты**

• **xAI анонсировала Grok Build на следующую неделю** — coding agent от xAI с режимом Arena: несколько AI-агентов решают одну и ту же задачу параллельно, пользователь выбирает победителя. Подход напоминает A/B-тестирование для кода — интересная эргономика, которую конкуренты не предлагают.
  ↳ https://blog.mean.ceo/new-ai-model-releases-news-april-2026/

• **Cloudflare предложила enterprise-стандарт для MCP-деплоя** — компания выпустила reference architecture: MCP-серверы должны быть remote (не local), управляться централизованной командой, иметь governance и cost controls. Введён «Code Mode», сворачивающий сотни tool definitions в несколько dynamic entry points — декларируют снижение token usage на 99.9%. Если Cloudflare продавит этот стандарт, enterprise MCP-деплой сильно изменится.
  ↳ https://www.infoq.com/news/2026/04/cloudflare-mcp/
  ↳ https://thenewstack.io/model-context-protocol-roadmap-2026/

• **Claude Code получил Vim visual mode и кастомные темы** — в апрельских обновлениях: `v` и `V` для visual/visual-line режимов с selection и операторами; `/cost` и `/stats` объединены в `/usage`; `/theme` позволяет создавать и переключать собственные темы прямо из CLI.
  ↳ https://code.claude.com/docs/en/whats-new

Параллельно AI-ассистент Meta для рекламодателей снизил cost per result у малого бизнеса на 12% и решал проблемы с аккаунтами на 20% лучше в beta. Компания делает ставку на то, что AI даст ей рекламное преимущество, не требующее роста штата.

Вопрос не в эффективности, а в том, кого именно выбирает модель — и есть ли у граждан способ это оспорить.

  ↳ https://www.transparencycoalition.ai/news/ai-legislative-update-april24-2026

**💡 Инсайт дня**

Две истории 25 апреля — OpenClaw и Cloudflare MCP — кажутся противоположными, но говорят об одном. OpenClaw вирусен, потому что пользователи хотят AI у себя: без облака, без подписок, без слежки. Cloudflare выпускает enterprise-стандарт, потому что корпорации хотят AI под контролем: governance, cost controls, centralized management. AI разошёлся в два полюса — и оба растут одновременно. Компании, которые строят только для одного полюса, рискуют пропустить другой. Тот же паттерн с OpenAI workspace agents (корпоративный) и OpenClaw (личный) — рынок не выбирает, он берёт оба.
