🗞 **AI Daily Digest — 13 апреля 2026**

**🔥 Главное**

• **Stanford AI Index 2026: Anthropic впервые №1, Китай почти догнал, публика в тревоге** — Ежегодный отчёт Stanford HAI вышел 13 апреля и задал тон дискуссии на весь день. Главные цифры: Anthropic занял первое место в рейтинге моделей (Claude Opus 4.6 опережает ближайшего конкурента лишь на 2,7%), xAI и Google делят второе-третье место, OpenAI — четвёртый. Китайские модели сократили отставание до минимума — американские и китайские системы несколько раз менялись местами на вершине бенчмарков с начала 2025 года. При этом индекс прозрачности Foundation Model Transparency Index рухнул с 58 до 40 баллов: топовые лабы всё реже раскрывают параметры обучения, размеры датасетов и архитектурные детали. Обратная сторона гонки — рост общественной тревоги: эксперты и широкая публика расходятся во взглядах сильнее, чем когда-либо.

**🤖 Claude & Anthropic**

• **Claude Mythos Preview: подробности Project Glasswing** — После недели туманных утечек Anthropic опубликовала детали: Mythos Preview автономно нашёл тысячи zero-day уязвимостей в каждой major ОС и каждом крупном браузере. Ключевой кейс — самостоятельная эксплуатация 17-летней RCE-уязвимости во FreeBSD (CVE-2026-4747), дающей полный контроль над сервером. На стресс-тестах Mythos успешно строил exploit-цепочки в 181 случае из нескольких сотен попыток; Opus 4.6 показывал ~0%. В Project Glasswing вошли ~40 компаний: Amazon Web Services, Apple, Broadcom, Cisco, CrowdStrike, Google, JPMorganChase, Microsoft, Nvidia. Широкого релиза не будет — Anthropic намерена удерживать модель под контролем вплоть до выработки industry-стандартов защиты.
  ↳ https://www.anthropic.com/glasswing
  ↳ https://red.anthropic.com/2026/mythos-preview/
  ↳ https://www.iowapublicradio.org/news-from-npr/2026-04-13/is-anthropics-claude-mythos-really-a-cybersecurity-bombshell

• **Claude теперь в Word — Microsoft Office трилогия завершена** — Anthropic выпустила Claude for Word Add-in, замкнув нативную интеграцию во весь классический офисный триод: Word, Excel, PowerPoint. Корпоративные пользователи получают Claude прямо в документах без переключения контекста.
  ↳ https://www.openpr.com/news/4468672/anthropic-rolls-out-claude-for-word-add-in-now-full-microsoft

• **Массовые 500-ошибки и споры о кеше** — 13 апреля сотни пользователей столкнулись с intermittent HTTP 500 на claude.ai, через API и в Claude Code — статусная страница при этом показывала «All Systems Operational». Параллельно The Register выяснил: Anthropic тихо сократил TTL кеша Claude Code с одного часа до пяти минут. Компания настаивает, что это не должно увеличивать затраты, но пользователи фиксируют аномально быстрое сгорание квот.
