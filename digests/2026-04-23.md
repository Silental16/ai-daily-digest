🗞 **AI Daily Digest — 23 апреля 2026**

**🔥 Главное**

• **OpenAI выпускает GPT-5.5 — «последний рубеж перед AGI»** — 23 апреля OpenAI без предупреждения выкатила GPT-5.5, полностью переобученную агентическую модель. Benchmarks: 60 баллов на Artificial Analysis Intelligence Index (на 3 пункта выше Claude Opus 4.7 и Gemini 3.1 Pro), 85,0% на ARC-AGI-2 (+11,7 над GPT-5.4), 82,7% на Terminal-Bench 2.0, 73,1% на Expert-SWE. Sam Altman назвал GPT-5.5 «последним значимым milestone перед AGI». Доступна сегодня для Plus/Pro/Business/Enterprise в ChatGPT и Codex. Цена API: $5/$30 за 1M токенов (вдвое дороже GPT-5.4), но экономия на output-токенах сводит реальный рост цен к ~20%.
  ↳ https://www.marktechpost.com/2026/04/23/openai-releases-gpt-5-5-a-fully-retrained-agentic-model-that-scores-82-7-on-terminal-bench-2-0-and-84-9-on-gdpval/
  ↳ https://tech.yahoo.com/ai/chatgpt/articles/openai-releases-gpt-5-5-192516262.html

• **Anthropic публично признаёт: три бага сделали Claude хуже** — The Register и VentureBeat опубликовали итоги внутреннего расследования Anthropic. Три изменения между мартом и апрелем незаметно деградировали качество Claude Code, Agent SDK и Cowork: (1) 4 марта понизили дефолтный reasoning effort с high на medium, чтобы убрать задержку в UI; (2) 26 марта кеширующий баг начал сбрасывать сессионный кэш на каждом turn; (3) 16 апреля добавили в system prompt лимиты <25 слов между tool calls и <100 слов в финальном ответе — кодинг-качество упало на 3%. В качестве компенсации Anthropic сбросил usage limits для всех подписчиков.

Акции конкурента RELX упали на фоне новости. Это сигнал: enterprise-юристы выбирают Claude, а не GPT.

• **Claude Code 23 апреля: vim-режим, /usage, темы и MCP в хуках** — свежий релиз добавил: visual mode (v/V) для vim с selection/operators; команды `/cost` и `/stats` объединены в `/usage`; создание и переключение именованных тем через `/theme` (и ручное редактирование JSON в `~/.claude/themes/`); хуки теперь могут вызывать MCP-инструменты напрямую через `type: "mcp_tool"`; новый env var `DISABLE_UPDATES` полностью блокирует все update-пути; WSL на Windows теперь умеет наследовать Windows-управляемые настройки через `wslInheritsWindowsSettings`. Фиксы: MCP OAuth, Homebrew install, производительность Write tool.
  ↳ https://github.com/anthropics/claude-code/releases
  ↳ https://code.claude.com/docs/en/changelog

**💡 Инсайт дня**

23 апреля 2026 вышло редкое совпадение: OpenAI торжественно выкатила «последнюю модель перед AGI», а Anthropic в тот же день публично расписалась в том, что три месяца незаметно ухудшала свой продукт внутренними оптимизациями. Это учебный кейс о том, как по-разному воспринимается качество модели: пользователи заметили деградацию в Claude Code мгновенно (Issue #41930 на GitHub), но Anthropic потребовалось несколько недель и публичное давление, чтобы опубликовать post-mortem. GPT-5.5 сейчас лидирует в benchmarks — но конкурентная гонка во многом решается не в benchmark-таблицах, а в ежедневном trust-балансе: насколько стабильно ведёт себя модель в production. Anthropic сделал правильное: признал вину, сбросил лимиты, дал объяснение. OpenAI сделал большое: выпустил новую SOTA-модель. Оба хода важны, но в разных временных горизонтах.
