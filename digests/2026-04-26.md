🗞 **AI Daily Digest — 26 апреля 2026**

**🔥 Главное**

• **Anthropic публично признала три инженерных ошибки, убившие Claude Code на месяц** — Fortune вышел с заголовком "Anthropic says engineering missteps were behind Claude Code's monthlong decline after weeks of user backlash". Три независимых бага накопились с марта по апрель: 4 марта понизили reasoning effort с «high» до «medium» ради скорости; 26 марта баг в кешировании начал сбрасывать историю рассуждений посреди сессии — модель казалась «забывчивой»; 16 апреля в system prompt добавили ограничение в 25 слов между вызовами инструментов. Все три исправлены к 20 апреля, 23 апреля Anthropic сбросил usage limits подписчикам. API при этом не был затронут ни разу.
  ↳ https://fortune.com/2026/04/24/anthropic-engineering-missteps-claude-code-performance-decline-user-backlash/
  ↳ https://www.theregister.com/2026/04/23/anthropic_says_it_has_fixed/

**🤖 Claude & Anthropic**

• **Claude Opus 4.7 — новый дефолт для Claude Code с 23 апреля** — модель вышла 16 апреля с 1M-токенным контекстом и high-resolution vision, а через неделю стала стандартной для всех Claude Code-сессий. На SWE-bench Pro: 64.3% — лучший результат среди всех моделей на момент релиза. Ровно через неделю OpenAI ответил GPT-5.5: 58.6% на SWE-bench Pro, 82.7% на Terminal-Bench 2.0 и омниmodal-архитектура (текст + изображения + аудио + видео). Гонка перешла в недельный ритм.
  ↳ https://www.digitalapplied.com/blog/ai-coding-assistants-april-2026-cursor-copilot-claude
  ↳ https://llm-stats.com/models/gpt-5.5

• **Claude Code апрельские обновления: Vim, темы, MCP hooks, Ultraplan** — помимо исправлений, апрель принёс: visual Vim mode (`v` и `V`) с selection и операторами; `/theme` для создания кастомных тем; прямые MCP tool hooks из hooks-конфига; Monitor tool для стриминга событий фоновых процессов; Ultraplan в early preview — черновик плана в облаке, редактирование в веб-редакторе; limit для MCP-инструментов поднят до 500K символов.
  ↳ https://code.claude.com/docs/en/whats-new
  ↳ https://help.apiyi.com/en/claude-code-changelog-2026-april-updates-en.html

**⚡ Инструменты & Агенты**

• **DeepSeek V4 — открытая модель 1M контекста, Apache 2.0** — вышла 24 апреля в двух вариантах: V4-Pro (1.6T параметров, 49B активируются, MoE-архитектура) и V4-Flash (284B параметров, 13B активируются). Оба поддерживают 1M-токенный контекст. DeepSeek-V4-Pro обходит все конкурирующие open-source модели по математике и коду, уступая лишь Gemini 3.1-Pro в world knowledge. Открытая лицензия Apache 2.0 — значит, модель можно скачать, запустить локально и модифицировать без ограничений.
  ↳ https://www.cnbc.com/2026/04/24/deepseek-v4-llm-preview-open-source-ai-competition-china.html
  ↳ https://huggingface.co/deepseek-ai/DeepSeek-V4-Pro

• **MCP-уязвимость: RCE «по замыслу» в официальном SDK Anthropic** — исследователи безопасности зафиксировали критический изъян в архитектуре MCP, позволяющий выполнять произвольный код на любой системе с уязвимой реализацией. Баг встроен в официальный MCP SDK Anthropic на нескольких языках, затрагивает более 200 000 публично доступных серверов и пакеты с суммарно 150M+ загрузок. Cloudflare предлагает enterprise-референс с «Code Mode», который сворачивает сотни tool definitions в динамические entry points и снижает token usage на 99.9%.
  ↳ https://thehackernews.com/2026/04/anthropic-mcp-design-vulnerability.html
  ↳ https://www.infoq.com/news/2026/04/cloudflare-mcp/
