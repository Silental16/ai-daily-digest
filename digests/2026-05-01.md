🗞 **AI Daily Digest — 1 мая 2026**

**🔥 Главное**

• **Пентагон подписал AI-соглашения с 8 компаниями — Anthropic за бортом** — Министерство обороны США допустило SpaceX, OpenAI, Google, NVIDIA, Microsoft, Amazon Web Services, Reflection и Oracle к засекреченным сетям уровней IL6 и IL7. Anthropic намеренно исключён: компания отказалась принять оговорку о «всех законных целях», включающую автономное оружие и массовую слежку. Ирония: два месяца назад Claude был единственной большой языковой моделью, одобренной на засекреченных сетях Пентагона в промышленном масштабе — теперь Anthropic вычеркнут по имени, а пустоту заполнили конкуренты.

**🤖 Claude & Anthropic**

• **Claude for Creative Work: коннекторы для творческих профессионалов** — Anthropic запустил набор коннекторов, превращающих Claude в управляющий слой для профессионального ПО: Autodesk Fusion (3D-моделирование через natural language), Ableton (управление музыкальным продакшеном), Blender (3D через промпты), Resolume Arena и Wire (live-визуалы в реальном времени), SketchUp (архитектурное 3D). Первое системное движение компании в сторону creative professionals — дизайнеров, архитекторов, музыкантов.
  ↳ https://www.anthropic.com/news/claude-for-creative-work
  ↳ https://all3dp.com/4/claude-comes-to-cad-anthropics-ai-can-now-help-build-3d-models-in-autodesk-fusion/

**⚡ Инструменты & Агенты**

• **Claude Code 2.1.126: критический фикс контекста Opus 4.7** — Обнаружен и исправлен серьёзный баг: Claude Code считал использование контекста Opus 4.7 по 200K-окну вместо нативного 1M. Из-за этого сессии автоматически компактировались в 5× раньше нужного. Также в релизе: новая команда `claude project purge` для полной очистки состояния проекта (транскрипты, задачи, история файлов) с поддержкой `--dry-run`; OAuth-аутентификация через вставку кода прямо в терминал (критично для WSL2, SSH и контейнеров); улучшен /resume для крупных сессий (до +67% скорости на >40 МБ).
  ↳ https://code.claude.com/docs/en/changelog

• **Microsoft Agent 365 вышел в релиз** — 1 мая Microsoft запустил Agent 365 ($15/user/month) — выделенную control plane для управления enterprise AI-агентами на Microsoft Foundry, Copilot Studio и сторонних платформах. IT-команды получают видимость и контроль над всеми автономными агентами в организации. Одновременно анонсирован E7 «Frontier Suite» за $99/user/month, объединяющий E5, Copilot, Agent 365 и Entra Suite.
  ↳ https://datanorth.ai/blog/top-10-ai-tools-for-2026

**🌐 Индустрия**

• **Карпати: вайб-кодинг уступает место агентной инженерии** — На конференции AI Ascent 2026 Андрей Карпати (основатель Eureka Labs) в разговоре с партнёром Sequoia Стефани Жан объяснил эволюцию парадигмы: vibe coding — это «разговор с AI о коде», agentic engineering — это «делегирование AI целых workflow». 92% американских разработчиков уже ежедневно используют AI-инструменты. Вопрос сместился с «использовать или нет» к «насколько глубоко доверять автономии».
  ↳ https://www.franksworld.com/2026/05/01/andrej-karpathy-on-the-evolution-from-vibe-coding-to-agentic-engineering/

• **Simon Willison: инструмент для iNaturalist — с телефона, в лесу** — Саймон Виллисон (автор Datasette) в майский уик-энд создал Python CLI для агрегации наблюдений iNaturalist прямо из кемпинга — используя Claude Code для web на телефоне. Результат: рабочий git-scraping репозиторий, обновляющий clumps.json в автоматическом режиме. Хороший кейс: Claude Code для web уже достаточно мощный для создания реальных инструментов без компьютера.

**💡 Инсайт дня**

1 мая 2026 — день, когда AI-индустрия разделилась по принципному водоразделу. Восемь компаний — от OpenAI до SpaceX — подписались под «AI-первой боевой силой» Пентагона. Anthropic отказался и оказался единственным крупным американским AI-игроком за забором. Параллельно та же Anthropic запускает коннекторы для Ableton и Blender и исправляет баг в Claude Code — то есть строит коммерческое будущее вдали от оборонки. В краткосрочной перспективе это выглядит как упущенный доход. В долгосрочной — может оказаться дифференциатором: единственный frontier-lab, у которого есть честный ответ на вопрос «а вы оружие делаете?». Насколько дорого это стоит в корпоративных продажах — узнаем по следующим раундам финансирования.
