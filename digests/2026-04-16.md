🗞 **AI Daily Digest — 16 апреля 2026**

**🔥 Главное**

• **Anthropic выпустил Claude Opus 4.7 — и признал, что Mythos его обходит** — Opus 4.7 официально стал самым мощным публично доступным Claude: заметный скачок в software engineering (особенно на сложных задачах), улучшенное vision, более вкусные и аккуратные профессиональные выходы. Цена не изменилась — $5/$25 за млн токенов. Доступен через API, Bedrock, Vertex AI, Microsoft Foundry. Но в релизных нотах Anthropic прямо написал: Opus 4.7 намеренно обучался с дифференциально ослабленными кибернетическими capabilities, чтобы не дотягиваться до уровня засекреченного Mythos. Для тех, кому нужен полный cyber-доступ — новая Cyber Verification Program.
  ↳ https://www.cnbc.com/2026/04/16/anthropic-claude-opus-4-7-model-mythos.html

• **MCP: критическая уязвимость в 150 млн загрузок** — Исследователи Ox Security раскрыли «системный дефект» в официальных MCP SDK для Python, TypeScript, Java и Rust. Фlaw позволяет выполнять произвольные команды, получая доступ к базам данных, API-ключам, историям чатов и внутренним системам. По данным Ox, это architectural design decision — уязвимость унаследует любой разработчик, строящий на MCP. Anthropic ответил: «STDIO execution model — это безопасный дефолт, ответственность за санитизацию лежит на разработчике.» Сообщество разделилось.
  ↳ https://www.infosecurity-magazine.com/news/systemic-flaw-mcp-expose-150/

**🤖 Claude & Anthropic**

• **Opus 4.7: детали релиза** — Модель показала существенный прирост на «наиболее сложных» coding-задачах по сравнению с Opus 4.6. Более сильные vision-способности плюс улучшенный «вкус» при генерации интерфейсов, слайдов и документов. Anthropic вводит Cyber Verification Program для security-профессионалов — те смогут получить доступ к более мощным cyber-capabilities через проверку личности.
  ↳ https://www.axios.com/2026/04/16/anthropic-claude-opus-model-mythos

• **Claude Code: апдейт с /tui, push-уведомлениями и focus-режимом** — Ключевые новинки: команда `/tui` и fullscreen-рендеринг без мерцания в рамках одного разговора; push notifications на мобилку когда Remote Control + «Push when Claude decides» активны; Ctrl+O теперь переключает verbose transcript; новая команда `/focus`; конфиг `autoScrollEnabled` для отключения автоскролла в fullscreen-режиме. Фиксы для MCP, editor actions, sessions и производительности.
  ↳ https://releasebot.io/updates/anthropic/claude-code

• **Identity verification на Claude** — Anthropic начал постепенный роллаут верификации личности пользователей «для нескольких сценариев использования». Пользователи встречают prompt при доступе к определённым capabilities. Детали — пока скудные.
  ↳ https://www.engadget.com/ai/anthropic-will-ask-claude-users-to-verify-their-identities-for-a-few-use-cases-115754092.html
