**🤖 Claude & Anthropic**

• **Claude Code Routines: переиспользуемые наборы команд доступны на всех платных планах** — Anthropic запустила Routines — фичу, позволяющую сохранять и переиспользовать последовательности команд, настроек и инструкций как именованные «рутины». Работает оффлайн на Mac. Вместе с этим в релизе: переключение worktree через EnterWorktree, PreCompact hook с возможностью блокировать компакцию (exit 2 или `{"decision":"block"}`), фоновые monitors для плагинов, alias `/proactive` → `/loop`. Стриминг больше не зависает — abort через 5 минут без данных с fallback на non-streaming.
  ↳ https://9to5mac.com/2026/04/14/anthropic-adds-repeatable-routines-feature-to-claude-code-heres-how-it-works/
  ↳ https://blockchain.news/ainews/claude-code-routines-launch-latest-update-now-available-across-all-paid-plans-features-docs-and-business-impact

• **Project Epitaxy: что Anthropic строит для Claude Code Desktop** — Утечки и инсайдерские отчёты раскрывают облик будущего редизайна Claude Code. Coordinator Mode превращает агента в оркестратора: планирует задачи, назначает их параллельным суб-агентам, собирает результаты.
